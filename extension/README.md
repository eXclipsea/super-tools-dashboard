## Installation

### From Chrome Web Store
1. Visit the Chrome Web Store
2. Search "Super Tools - AI Productivity Suite"
3. Click "Add to Chrome"

### Developer Installation
1. Clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the extension folder

## Permissions

The extension requests these permissions:

- **activeTab**: Access current tab content
- **storage**: Save user preferences and analytics
- **scripting**: Inject functionality into web pages
- **contextMenus**: Add right-click menu options
- **sidePanel**: Create persistent side panel

## Privacy

- All data processing happens locally
- No personal data sent to external servers
- Usage analytics stored locally only
- Optional cloud sync for cross-device access
